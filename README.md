
# Back-end API 

### Esta aplicação é um modelo utilizada nas aulas de Web do IFPI Campus Picos pelo Prof. Jesiel Viana

## Tecnologias utilizadas
- Javascript
- Node.js
- Express
- Jest

## Instalação

```bash
$ git clone https://github.com/jesielviana/backend-aula-api.git
$ cd backend-aula-api
$ npm install
```

## Rodar o app

```bash
# development
$ npm start
```

## Testar

```bash
# run tests
$ npm run test
```
